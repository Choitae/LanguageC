	정수 입력(0 이상 29 이하의 정수),
	"0이상 10미만" / "10이상 20미만" / "20이상 30미만"

switch(변수) {				
case 1:					
	...				
	break;				
case 2:
	...
	break;
case 3:
	...
	break;
}					
---> switch 괄호 안의 변수의 값에 해당하는 case문을 찾아간다.
     case문의 작성, 실행할 내용을 적고 마지막으로 break; 로 마무리.
     switch문 안에서 break; 를 만나면, switch문을 탈출한다.
