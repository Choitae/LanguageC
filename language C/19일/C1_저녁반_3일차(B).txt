논리연산자 : &&  ||  !  -----------> 참과 거짓(즉, 조건식)에 대한 연산.
				     (NOT 숫자에 대한 연산)

	(ex) "빨간 사과가 하나 있다"
	     ----> (사과가 빨갛냐) && (사과가 하나냐)
		          ==                 ==

&& : AND연산. 둘 다 참일 때에만 참, 아니면 거짓.
	참 && 참 --------> 참
	참 && 거짓 ------> 거짓
	거짓 && 참 ------> 거짓
	거짓 && 거짓 ----> 거짓

|| : OR연산. 둘 중 하나라도 참이면 참, 아니면 거짓.
	거짓 || 거짓 ----> 거짓
	참 || 거짓 ------> 참
	거짓 || 참 ------> 참
	참 || 참 --------> 참

! : NOT연산. 뒤집어.
	! 거짓 ----------> 참
	! 참 ------------> 거짓
-----------------------------------------------------
(ex) 	int n = 12;
	n>10 --------> 참
	n%2==0 ------> 참
	n>10 && n%2==0 ---> 참 && 참 ---> 참
					  (의미: n이 10보다 큰 짝수냐)

조건식 연습:
1)	"a가 5보다 크고 10보다 작냐"
	----> (a가 5보다 크냐) 그리고 (a가 10보다 작냐)
	----> a>5 && a<10

2)	"a가 10보다 작은 짝수냐"
	----> (a가 10보다 작냐) 그리고 (a가 짝수냐)
	----> a<10 && a%2==0

3)	"a가 10보다 크거나 5보다 작냐"
	----> (a가 10보다 크냐) 또는 (a가 5보다 작냐)
	----> a>10 || a<5

	(ex) int num = 5;
	     printf("%d", -num);     // -5  --> num의 값이 바뀐 것은 아님.
	     printf("%d", num);      // 5
	     ----> 변수의 값을 바꾸려면, 대입연산자(=) 또는 증감연산자(++,--).

삼항연산자( ? : )
	"조건식 ? 참일때 : 거짓일때"
	    A   ?    B   :    C

	(ex)	int num = 5;
		char c = (num>0 ? '+' : '-');
		printf("%c", c);     // +

	     -------> 0과 1에 대한 연산.

비트논리연산자
	& (비트and연산자) : 둘 다 1인 경우에만 1.
	| (비트or연산자) : 둘 중 하나라도 1인 경우에는 1.
	~ (비트not연산자) : 뒤집어.
	(ex) 0011 & 0001 = 1 (0001)
	(ex) 0011 | 0001 = 3 (0011)
	(ex) ~3 ---> ~0011 = 1100

제어문(Control Statement) : 조건문, 반복문

조건문 : if문, switch문.

if문.
   기본:  if ( 조건식 ) {	"만약에 ~라면"
	      실행할 내용#1;
	  }
   추가:  else {		"그렇지 않으면"
	      실행할 내용#2;
	  }
   (이해) -------> 내용#1 실행되는 경우 : 조건식이 참일 때.
		   내용#2 실행되는 경우 : 조건식이 거짓일 때.

   -----------------------------------------------------------
   기본:  if ( 조건식1 ) {	"만약에 ~라면"
	      실행할 내용#1;
	  }
   추가:  else if ( 조건식2 ) {  "그게 아니라 ~라면"
	      실행할 내용#2;
	  }
   추가:  else {		"그것도 아니면"
	      실행할 내용#3;
	  }
   (이해) ------> 내용#1 실행되는 경우 : 조건식1 이 참일 때
		  내용#2 실행되는 경우 : 조건식1 이 거짓이고,
					 조건식2 가 참일 때
		  내용#3 실행되는 경우 : 조건식1,2가 모두 거짓일 때.

	[실수 주의] if ( 조건식1 ) { ... }
		    else ( 조건식2 ) { ... }  ------------ X
